## json-server
可以开启一个数据接口。
### 启动
`json-server --watch data.json`
## http-server
使用 Node.js 搭建http服务器 http-server 模块
### 安装 http-server 模块
`npm install http-server -g`
### 启动
1. cd 到需要启动 http-server 服务的文件夹
1. `http-server -c-1` 或者 `http-server -c-1 -p 8081 `
> http-server  默认的端口是8080    可以使用 -p 参数指定一个端口号
